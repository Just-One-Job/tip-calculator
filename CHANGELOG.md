# Changelog

All notable changes to the Tip Calculator project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-XX

### Added
- Initial release of Tip Calculator
- Bill amount input with numeric keypad
- Tip percentage selector with presets and custom option
- Split bill functionality (1-10+ people)
- Real-time calculation of tip amount, total with tip, and per-person total
- Dark mode support with toggle switch
- Persistent storage for tip percentage and split count preferences
- Haptic feedback on all button interactions
- Theme system with light and dark color schemes
- Responsive, thumb-friendly button sizes
- Clean, one-screen interface
- No ads, tracking, or analytics
- TypeScript support with strict mode
- Comprehensive documentation (README, CONTRIBUTING, SETUP)

### Features
- **Core Functionality**
  - Numeric keypad for bill input
  - Quick tip percentage selection
  - Custom tip percentage input
  - Bill splitting between multiple people
  - Real-time calculation updates

- **User Experience**
  - Dark/light mode toggle
  - Haptic feedback
  - Persistent preferences (tip % and split count)
  - Large, accessible touch targets
  - Smooth animations and transitions

- **Design**
  - Clean, minimal interface
  - One-screen design philosophy
  - Privacy-first (no tracking)
  - Fast launch time
  - Accessible typography

### Technical Details
- Built with React Native and Expo
- TypeScript for type safety
- AsyncStorage for local persistence
- Expo Haptics for tactile feedback
- React Native Safe Area Context for proper spacing
- Component-based architecture
- Theme context for consistent styling
